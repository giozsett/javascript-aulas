Aula 10 - DOM

Eventos DOM de mouse:

  • mouseenter: Cursor do mouse foi movido até dentro do elemento.
  • mousemove: Cursor do mouse foi movido enquanto já estiver dentro do elemento.
  • mousedown: Cursor do mouse foi pressionado dentro do elemento.
  • mouseup: Cursor do mouse foi solto após ter sido pressionado dentro do elemento.
  • click: Evento que corresponde ao clique "inteiro".
  • mouseout: Quando o cursor do mouse for movido para fora do elemento.

Função

Consiste no conjunto de linhas de código que serão executadas quando um determinado evento ocorrer.
Uma função não é executada automaticamente, ela precisa de um evento para que seja "ativada".
A função deve estar dentro de um bloco que é delimitado pelos sinais { }
Função Anônima: O JavaScript possui algumas funções que não têm nome, porém ainda assim, uma função precisa de um nome para ocorrer.
As funções são escritas da seguinte forma:
function ação(parâmetros) { }
