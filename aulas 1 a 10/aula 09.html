DOM = Document Object Model
DOM é o conjunto de objetos do navegador que dão acesso a componentes internos do website.


Árvore DOM

  • Window: É a raiz, tudo que está JavaScript está dentro desse objeto chamado Window.
  • Location: Localização e endereço do site, URL da página atual, qual foi página anterior, etc.
  • Document: Página atual. Dentro dela há o HTML.
  • History: Histórico de páginas, facilita a navegação dentro do site.

Formas de acessar um elemento dentro do DOM:

  • por Marca: getElementsByTagName()
  • por ID: getElementById() - esse seletor é no SINGULAR.
  • por Nome: getElementsByName() - só é possível utilizar quando há um atributo name.
  • por Classe: getElementsByClassName()
  • por Seletor: querySelector() e querySelectorAll() - Presente apenas nas versões mais recentes do ECMA script. Mais prático para alterações rápidas no site.
