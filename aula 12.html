Aula 12 - Condições (parte 02)

Condições Aninhadas

São condições compostas (com if e else) que possuem outras condições dentro delas.
Utilizam um elemento chamado else if.


Condição Múltipla

São utilizadas quando é necessário trabalhar com valores fixos. Testa vários valores e depois retorna ao fluxo natural do código.
No entanto, não são recomendadas para trabalhar com intervalos de valores.
Dentro de cada bloco precisa haver um comando break.

switch (expressão) {
    case valor1:

       break

    case valor2:

       break

    default:

       break
}
