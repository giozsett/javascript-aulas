Operadores Relacionais

  • > maior que
  • < menor que
  • >= maior ou igual
  • <= menor ou igual
  • == igual a
  • != diferente de
No Javascript, o sinal = é um sinal de atribuição e não um operador de igualdade, por isso que a igualdade é representada com ==.
Operadores Relacionais também podem ser usados com variáveis do tipo string. Por exemplo: curso == 'Javascript'

Ordem dos operadores:

  • 1º Operadores Aritméticos
  • 2º Operadores Relacionais
  • 3º Operadores Lógicos
  • 4º Operadores Ternários

Operadores de Identidade

Testam o tipo das variáveis. Em Javascript usar apenas o sinal de igualdade não irá testar, por exemplo, se uma variável é string ou number.
5 == 5 é true e 5 == '5' também será true
Para verificar o tipo de dado de uma variável é necessário usar os Operadores de Identidade da seguinte forma: 5 === 5


Operadores Lógicos

  • ! Negação (not). É considerado um operador unário por acompanhar apenas 1 operando.
  • && Conjunção (e/and)
  • || Disjunção (ou/or)
Ordem dos Operadores Lógicos:
  • 1º Não / Not
  • 2º E / And
  • 3º Ou / Or

Operadores Ternários

Servem para testar alguma condição.
Possui esse nome por ter 3 blocos: bloco de teste, bloco verdadeiro e bloco falso.
teste ? true : false
Correspondem aos if e else.
