Aula 11 - Condições (parte 01)

Os comandos costumam ser executados em sequências.
No entanto, há situações em que um desses comandos não deverá ser executado, é nesses casos que as Condições são utilizadas.
Também há situações em que há desvios no fluxograma de um programa, as Condições são essenciais para que isso se torne possível.


If -- estrutura:

if (condição) { }
else { }
Há os blocos para caso a condição seja verdadeira (if e elif) e o bloco para o caso da condição ser falsa (else).
Uma condição pode ou não ter o else, geralmente nas condições que não possuem essa parte, caso a condição verificada pelo if seja falsa, nada de especial ocorre e o código apenas segue seu fluxo.
