Aula 13 - Repetições

Sequências

Executa os comandos em uma sequência. As sequências são um exemplo de estrutura de repetição.


Laços de Repetição

Essas estruturas de repetição testam uma condição e criam um loop que testa a condição diversas vezes e se repetem enquanto a condição for verdadeira. Laços de repetição while possuem teste lógico no início.
Os laços de repetição while são escritos da seguinte forma:

while (condição) {
    bloco de código
}


Também há os laços de repetição do tipo faça, que são os laços de repetição com teste lógico no final.
Os laços desse tipo são escritos da seguinte forma:

do {
    bloco de código
} while (condição)
